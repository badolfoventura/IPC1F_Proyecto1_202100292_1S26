# Manuales del Proyecto 1

En esta carpeta se almacenan los documentos relacionados con el **Proyecto 1** del curso **Introducción a la Programación y Computación 1**.

## Contenido de la carpeta

Esta carpeta contendrá los siguientes documentos:

- **Manual Técnico**
  
  Documento que describe la estructura del sistema, los métodos implementados en el programa y los requerimientos necesarios para ejecutar la aplicación.

- **Manual de Usuario**
  
  Documento que explica cómo utilizar el programa, incluyendo la interacción del usuario con la interfaz gráfica.

- **Diagrama de Flujo**
  
  Representación gráfica del flujo lógico del funcionamiento del programa.

## Información del Proyecto

**Proyecto:** Sistema de gestión de productos  
**Lenguaje:** Java  
**Interfaz gráfica:** Java Swing  

El sistema permite gestionar productos mediante una interfaz gráfica que incluye funcionalidades como:

- Agregar productos
- Eliminar productos
- Registrar ventas
- Visualizar información del estudiante
- Navegación entre ventanas
